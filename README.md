# 🌐 JavaScript API Dashboard


---

## 🚀 Features
This dashboard includes:

- 🐶 Random Dog Image Generator
- 🐱 Random Cat Image Generator
- ☀️ Weather Information (Open-Meteo API)
- 💱 Currency Exchange Rates
- 🎬 Trending TV Shows
- 🧑‍💻 GitHub User Profile Viewer
- 🤣 Random Joke Generator
- 📚 Random Cat Facts

---

## 🛠️ Technologies Used
- HTML5
- CSS3 (Grid Layout + Styling)
- JavaScript (ES6)
- Fetch API
- async / await
- DOM Manipulation

---

## 🔧 How It Works
Each section of the dashboard makes a request to a public API using JavaScript's Fetch API.

When a button is clicked:
1. JavaScript sends a request to an API
2. The response is returned in JSON format
3. The data is extracted and displayed on the page
4. Only the specific section updates (no page reload)

Error handling is included using `try...catch` to prevent the app from breaking if an API fails.

---

## ▶️ How to Run the Project
1. Download or clone this repository
2. Open the project folder
3. Open `index.html` in your browser
4. Click the buttons to load data from the APIs
